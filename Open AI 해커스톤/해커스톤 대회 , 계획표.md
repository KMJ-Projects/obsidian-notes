### **1. 대회 개요**

- **대회명:** 조코딩 x OpenAI x 프라이머 AI 해커톤 
- **주최/후원:** 조코딩, OpenAI, 프라이머(Primer), 한빛미디어
- **주제:** OpenAI의 API를 한 개 이상 활용하여 **비즈니스 가치가 있는 프로덕트(서비스)** 만들기 
- **참가 대상:** 누구나
- **참가 형태:** 1인 혹은 팀 단위

### **2. 주요 일정**

- **예선 접수 기간:**  ~ 2026년 2월 20일
- **예선 결과 발표:** 3월 2일
- **본선 및 시상식:** 3월 7일 (토요일)

### **3. 예선 접수 및 심사 기준**

- **제출물:** 실제로 작동하는 프로덕트, 서비스 소개 및 작동 시연 영상
- **주요 심사 기준:**
    - **비즈니스 가치:** 단순히 API를 무료로 개방하는 것이 아니라, 수익 모델(결제 시스템 등)이나 지속 가능성이 있는지가 중요합니다.
    - **작동 여부:** 시연 영상을 통해 실제 작동을 확인합니다 

### **4. 참가 및 수상 혜택**

- **예선 참가 혜택 (기준 만족 시 전원):** 약 45만 원 상당의 혜택 (OpenAI API 크레딧 및 ChatGPT Pro 1개월 이용권 등 - 현재 협의 중이나 긍정적)
- **본선 진출 혜택 (30팀 선발):**
    - 본선 행사 초대 (저녁 식사 및 네트워킹 제공) 
    - OpenAI 굿즈, 도서, 음료 등 제공 
- **최종 우수 팀 혜택:**
    - **투자 기회:** 프라이머를 통해 우수 팀에게 **1억 원 규모의 투자** 진행 및 사업화 멘토링 제공   - 추가적인 OpenAI 크레딧 시상 (협의 중)

### 추천 기술 스택

| 영역    | 기술                              | 이유                    |
| ----- | ------------------------------- | --------------------- |
| 프론트엔드 | Next.js 14 + TypeScript         | SSR + API Routes 한 번에 |
| 백엔드   | Python FastAPI                  | OpenAI SDK 최적, 빠른 개발  |
| DB    | Supabase (PostgreSQL)           | 무료, Auth 내장, 실시간 구독   |
| AI    | OpenAI API (GPT-4o, DALL-E 3 등) | 대회 필수                 |
| 배포    | Vercel (프론트) + Railway (백엔드)    | 무료 티어, 빠른 배포          |
| 결제    | Toss Payments or Stripe         | 시연용 테스트 결제            |
