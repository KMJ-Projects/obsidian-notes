## 📌 제품/서비스명 

HairFlow - AI 헤어 시술 레시피 & 미래 예측

## 📌 한 줄 소개 

고객 사진 한 장으로 시술 레시피부터 8주 후 변화까지 예측하는 헤어 디자이너 전용 AI

---

## 📌 상세 설명

### 문제

전국 헤어 디자이너 약 30만명. 이들이 매일 겪는 세 가지 치명적 문제가 있습니다.

**1) 시술 실패의 공포.** 고객이 레퍼런스 사진을 들고 옵니다. "이렇게 해주세요." 하지만 고객의 실제 모질, 손상도, 현재 베이스 컬러는 사진 속 모델과 전혀 다릅니다. 경력 3년차 디자이너는 감으로 약제를 배합하고, 결과가 다르게 나오면 재시술에 클레임까지. 시간, 약제, 신뢰 모두 잃습니다.

**2) 고객이 다시 안 오는 문제.** 아무리 잘 해줘도 고객은 "다음에 올게요~" 하고 나가면 잊어버립니다. 염색은 6주면 퇴색되고, 펌은 3개월이면 풀리는데, 재방문 타이밍을 알려주는 사람이 없습니다. 결국 다른 미용실로 갑니다. 평균 재방문율 40%. 나머지 60%는 매출 증발입니다.

**3) 서로 모호한 상황**. 남자 고객들이 미용실 가서 제일 많이 하는 멘트는 "깔끔하게 잘라주세요."  
고객과 헤어 디자이너 모두 모호한 상황속에서 커트가 진행됩니다.  
이런일이 발생하지 않도록 고객 맞춤형 헤어 분석 및 커트가 진행되어야 합니다.

### 해결책

**[STEP 1] AI 시술 레시피 (시술 전)**

고객 레퍼런스 사진 + 고객의 현재 모발 사진을 업로드하면, OpenAI GPT-4o Vision이 두 사진을 정밀 비교 분석합니다.

- 현재 모발 상태 진단: 베이스 레벨, 손상도, 모질, 모량
- 목표 스타일과의 차이 분석: 컬러 갭, 길이 차이, 텍스처 차이
- 맞춤 시술 레시피 카드 자동 생성:
    - 컬러: 약제 브랜드별 배합 비율, 도포 순서, 방치 시간
    - 커트: 섹션별 길이, 각도, 레이어 기법
    - 펌: 롯드 사이즈, 와인딩 방향, 약제 처리 시간
    - 주의사항: "후두부 기존 염색 잔여분 있음, 프리라이트닝 필요"

경력 3년차 햄스터 디자이너라도 이 레시피만 따라가면 10년차 수준의 시술이 가능합니다.

**[STEP 2] AI 미래 예측 타임라인 (시술 후)**

시술 완료 사진을 업로드하면, AI가 시간 경과에 따른 변화를 예측합니다.

- 염색: 1주 / 2주 / 4주 / 6주 / 8주 퇴색 과정 이미지 생성
- 커트: 2주 / 4주 / 8주 자라나는 과정 이미지 생성
- 펌: 1주 / 4주 / 8주 / 12주 풀리는 과정 이미지 생성

고객에게 "4주 후엔 이렇게 변합니다. 그때 오시면 가장 좋아요"라고 보여주면 → 재방문 예약율 폭증

### 활용 OpenAI API

- GPT-4o Vision: 모발 상태 정밀 분석, 레퍼런스 vs 현재 비교
- DALL-E 3: 시간 경과별 모발 변화 예측 이미지 생성
- GPT-4o: 시술 레시피 텍스트 생성, 약제 배합 계산, 주의사항 도출

### 기술 스택

- Frontend: Next.js / TypeScript
- Backend: Python FastAPI
- DB: Supabase (PostgreSQL)
- AI: OpenAI GPT-4o Vision + DALL-E 3 + GPT-4o
- 배포: Vercel + Railway

### 왜 ChatGPT로는 불가능한가
ChatGPT에게 "이 고객 사진 보고 시술 레시피 알려줘"라고 할 수 있지만
헤어 디자이너 전용 앱서비스는 존재하지 않고 , 헤어디자이너가 ChatGPT쓰고있기는 짜침
- ChatGPT는 이전 고객 기록을 기억하지 못함.
- ChatGPT는 시간 경과에 따른 변화 예측 이미지를 생성하지 못함.
- ChatGPT는 한국 약제 브랜드별 배합 데이터가 없음
- ChatGPT는 고객별 시술 히스토리를 누적 관리하지 못함
- HairFlow는 헤어 전문 데이터로 최적화된 프롬프트 + 고객 DB + 이미지 생성이 하나의 시스템으로 연결

## 📌 고객 정의 및 해결 방안

### 고객 1

- 나의 고객은 어떤 문제점을 가진 누구인가? → 레퍼런스 사진대로 시술했는데 결과가 달라서 재시술과 클레임에 시달리는 경력 1~5년차 주니어 헤어 디자이너
- 그 고객의 문제를 무엇으로 해결할 것인가? → 고객 모발 상태를 AI가 정밀 분석하여 약제 배합, 시술 순서, 주의사항까지 포함된 맞춤 시술 레시피를 자동 생성. 감이 아닌 데이터로 시술하여 한 번에 원하는 결과 달성.

### 고객 2

- 나의 고객은 어떤 문제점을 가진 누구인가? → 시술 후 고객 재방문율이 낮아 매출이 정체된 헤어 디자이너 (재방문율 평균 40%)
- 그 고객의 문제를 무엇으로 해결할 것인가? → 시술 완료 사진에서 AI가 1주~8주 후 변화 예측 이미지를 생성. 고객에게 "이 시점에 관리하면 가장 좋습니다"를 시각적으로 보여줘 그 자리에서 재방문 예약 확보. 재방문율 40% → 80% 목표.

### 고객 3

- 나의 고객은 어떤 문제점을 가진 누구인가? → 주니어 디자이너 시술 품질이 들쭉날쭉해서 클레임과 고객 이탈이 발생하는 미용실 원장
- 그 고객의 문제를 무엇으로 해결할 것인가? → AI 시술 레시피를 매장 전 직원이 공유하여 시술 품질을 균일화. 원장이 직접 가르치지 않아도 주니어가 고품질 시술 가능. Pro/Enterprise 플랜으로 매장 전체 도입.
