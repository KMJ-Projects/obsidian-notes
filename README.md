# Obsidian 학습 노트 업로드 방법

## 업데이트 방법
Obsidian에서 내용을 수정하거나 추가한 후, 터미널에 다음 명령어로 GitHub에 업로드

### 1. Obsidian 폴더로 이동
```bash
cd ~/Desktop/Code/obsidian
```
### 2. 변경사항 추가
```bash
git add .
```
### 3. 커밋 메시지와 함께 저장
```bash
git commit -m "ex) Claude Code 사용법"
```
### 4. GitHub에 업로드
```bash
git push
```

## 사용 도구

- **Obsidian**: 노트 작성 및 관리
- **Git**: 버전 관리
- **GitHub**: 온라인 저장 및 공유
